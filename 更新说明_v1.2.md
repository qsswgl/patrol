# 巡更打卡 APP 更新说明 v1.2

## 📅 更新日期
2025年11月27日

## ✨ 全新界面设计

### 🎨 界面优化

#### 1. 紫色品牌标题栏
- **紫色渐变设计** (#6B46C1)
- 顶部固定显示 "巡更打卡"
- 更符合品牌视觉识别

#### 2. 重新设计的信息卡片
- **当前位置卡片**:
  - 位置以大字号紫色显示 (32pt)
  - 时间以粗体黑色显示 (20pt)
  - NFC 状态实时显示
  - 最后读卡信息展示
  - 圆角阴影设计,视觉层次更清晰

#### 3. 优化的同步提示
- **橙色警告样式** (#FFA500)
- 浅橙色背景 (#FFF7ED)
- 显示待同步数量
- 右侧"立即同步"按钮醒目
- 无待同步记录时自动隐藏

#### 4. 改进的打卡记录列表
- **卡片式设计**:
  - 每条记录独立卡片
  - 位置名称加粗显示 (18pt)
  - 时间格式: yyyy-MM-dd HH:mm:ss
  - NFC ID 灰色小字显示
  - 同步状态标签右上角显示

#### 5. 全新空状态提示
- 大号表情图标 (64pt)
- 友好的提示文字
- 渐变透明度设计
- 引导用户操作

## 🎯 设计细节

### 颜色方案
| 元素 | 颜色代码 | 用途 |
|------|---------|------|
| 主色调 | `#6B46C1` | 标题栏、位置文字 |
| 警告色 | `#FFA500` | 同步提示、立即同步按钮 |
| 文字灰 | `#64748B` | 标签文字 |
| 深色文字 | `#1E293B` | 时间显示 |
| 浅灰文字 | `#94A3B8` | NFC ID |
| 边框灰 | `#E2E8F0` | 卡片边框 |

### 字体大小
- 标题: 24pt (粗体)
- 位置: 32pt (粗体)
- 时间: 20pt (粗体)
- 记录位置: 18pt (粗体)
- 记录时间: 14pt
- NFC ID: 12pt
